VAR_ARIMA_210208.py : 주가 데이터를 예측하는 전통적 통계 모델 구현 (VAR, ARIMA 모델)

modelcheck.sh : 모델학습 후 저장된 가중치값을 이용하여 모델 성능 확인

stock_210202.py : LSTM 모델을 이용한 주가예측 (주식회전율 값을 추가 feature로 사용, 리눅스 쉘 스크립트에서 자동으로 실행되도록 구현)
